# 4. 他委員会との連携

<Card>
### 📢 広報委員会
- **動画公開:** 大会後2週間以内に動画をアップロードすることが好ましいです。
</Card>

<Card>
### 🤝 マーケティング委員会
- **協賛関連:** 大会の協賛関連で、体育館の設営時に連携が必要になる場合があります。
- イベントのヘルプや大会協賛関連の手伝いが発生する可能性があります。
</Card>

<Card>
### 👔 理事会
- **ルール変更:** ルールブックの改訂やルールの変更・修正が入る場合があるため、動向に注意してください。
</Card>
