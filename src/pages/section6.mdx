# 6. 当日運営・備品管理

<div id="sub-timeschedule"></div>

## ⏱ タイムスケジュールの目安

<Card>
| 項目 | 所要時間の目安 |
| :--- | :--- |
| 設営 ＋ 開会式 | **60分** |
| 入場 ＋ アップ（設営なしの場合） | **30分** |
| 試合時間 | 1イニング：**20分**<br /> <small style="color:#666;">3イニング → 60分 / 4イニング → 80分</small> |
| 休憩時間 | **10〜20分** |
| 片付け〜撤収完了 | **45〜60分** |
</Card>

> 💡 **スケジュール作成のコツ**
> 試合数・コート数・休憩回数をもとに、上記の目安時間を組み合わせてタイムスケジュールを組み立ててください。

## 🖊 ラインテープの必要量

> ⚽ 1コートの設営に **1.5〜2巻き（約50m）** が目安となります。

### 💡 コート数別の必要量の目安

<Card>
| コート数 | 必要なテープ量（目安） |
| :--- | :--- |
| 4面 | 6〜8巻き（約300m） |
| 6面 | 9〜12巻き（約450m） |

<small style="color:#888;">※ 予備も考慮して多めに準備することを推奨します。</small>
